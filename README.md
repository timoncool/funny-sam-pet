# Funny Sam Pet

Codex pet package for a playful pixel-art Sam Altman mascot.

![Funny Sam preview](preview.gif)

## What is included

- `pet.json`
- `spritesheet.webp`
- `preview.gif`

## Install into Codex

Copy the two files into your Codex pets directory:

```text
C:\Users\user\.codex\pets\funny-sam\
```

You should end up with:

- `C:\Users\user\.codex\pets\funny-sam\pet.json`
- `C:\Users\user\.codex\pets\funny-sam\spritesheet.webp`

## Notes

- `pet.json` uses `spriteVersionNumber: 2`.
- The spritesheet is the finalized v2 asset.
- If the pet list shows broken text, restart the app once after replacing the files.
